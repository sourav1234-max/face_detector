Face Detection

Control the global face detection pipeline. Changes take effect immediately — no server restart needed.

Face Detection is Enabled
All uploaded images are processed through the full detection pipeline.
Global Face Detection Toggle
Enable or disable face detection for all incoming uploads, Google Drive syncs and local syncs.
Enable Face Detection Pipeline When enabled, all uploaded images go through face detection, landmark extraction, descriptor generation and metadata storage.
When Face Detection is Disabled:
  • Uploaded images are stored normally (file + metadata)
  • No Python face detection models are loaded or invoked
  • No descriptors, landmarks or embeddings are generated
  • Photos are saved with empty descriptors: [] and status "Skipped (Detection Disabled)"
  • Face search results for those images will return zero matches
Manual Face Correction
Even when face detection is disabled at upload time, you can manually draw face bounding boxes and update descriptors for any individual photo in the Photo Moderation section.
